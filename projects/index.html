Current Projects

Here’s a quick glimpse into the projects I’m currently involved with.

LinkerBk

Obtaining executable files from the object files of the MACRO-11 assembler (BSD version of the Richard Krehbiel’s assembler). It understands almost everything:

  • Name, version and start address of module
  • Program sections, both absolute and relocatable
  • Complex links
  • etc.

It’s easier to point out that it does not understand: libraries: smile :.

ICO2Obj

ICO2Obj - utility for converting ICO files to MACRO-11 object files for subsequent linking to executable images. It is possible to create one atlas from several image and refer to them by name from the program.
